# WeraLink

A modern micro-gig marketplace for Kenyan youth.

## Project Status: PHASE 0 COMPLETE 🚀

This repository has been initialized with a robust, production-ready foundation.

### Frontend (`weralink-client`)

- **Framework**: React.js with TypeScript & Vite.
- **Styling**: Tailwind CSS v4 (using the `@tailwindcss/vite` plugin).
- **UI Library**: `shadcn/ui` with custom color variants.
- **Architecture**: Feature-Driven Architecture (Bulletproof React pattern).
- **Core Dependencies**: `react-router-dom`, `lucide-react`, `axios`, `react-hook-form`, `zod`, `zustand`, `@tanstack/react-query`.

### Backend (`weralink-server`)

- **Environment**: Node.js & Express.js.
- **ORM**: Prisma.
- **Database**: PostgreSQL (hosted on Supabase).
- **Auth Strategy**: JWT via Supabase Auth.
- **Schema Status**: Successfully pushed and synced with Supabase, featuring UUID-based security, M-Pesa transaction types, and audit timestamps.

### Source Control & Knowledge

- **Unified Git Repo**: Monolithic structure across client/server.
- **AI-Ready Documentation**: Detailed instructions, architectural guidelines, and project context are maintained in the `.agents/` folder.
